Coin Converter: Consumindo API de cotações de moedas.


Este é o meu primeiro projeto consumindo uma API em JavaScript, nele eu desenvolvi um conversor de moedas consumindo a API do AwesomeAPI.


O Coin Converter pode converter qualquer moeda para a nossa moeda nativa (BRL), utilizando uma API que está em constante atualização, nosso conversor sempre tem os valores atualizados das cotações de cada moeda, proporcionando uma maior fidelidade com os resultados das conversões!
